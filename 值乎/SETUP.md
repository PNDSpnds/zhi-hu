# 值乎 - 微信小程序搭建指南

## 一、准备工作

### 1. 下载微信开发者工具
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
选择稳定版（Windows）下载安装。

### 2. 打开项目
- 启动微信开发者工具
- 选择「导入项目」或「新建项目」
- 项目目录：选择本项目根目录（`值乎/`）
- AppID：`wx84f90634ffbf5346`（已配置在 project.config.json）
- 后端服务：选择「微信云开发」

---

## 二、开通云开发

1. 项目打开后，点击顶部工具栏「云开发」按钮（云朵图标）
2. 首次使用会弹出开通界面：
   - 环境名称：`值乎`（随意命名）
   - 套餐：选择「免费版」即可
   - 点击「确定」
3. 等待 1-2 分钟环境创建完成

### 获取环境 ID
开通后，在云开发控制台顶部可以看到「环境 ID」，类似 `zhihu-xxx`。
复制这个 ID。

### 配置环境 ID
打开 `miniprogram/app.js`，将第 8 行的 `env` 值替换为你的环境 ID：
```js
wx.cloud.init({
  env: '你的环境ID',  // 替换这里
  traceUser: true
});
```

---

## 三、创建数据库集合

在「云开发控制台」→「数据库」→ 点击「添加集合」，依次创建以下 6 个集合：

| 集合名 | 用途 |
|--------|------|
| `users` | 用户数据 |
| `evaluations` | 评测记录 |
| `posts` | 社区帖子 |
| `votes` | 投票记录 |
| `comments` | 评论 |
| `feedback` | 回馈记录 |

每个集合创建后，权限设置为「仅创建者可读写」（后面可改为自定义权限）。

---

## 四、上传部署云函数

在微信开发者工具左侧文件树中，找到 `cloudfunctions/` 目录。

**依次对每个云函数上传部署：**
1. 右键点击 `cloudfunctions/login` → 选择「上传并部署：云端安装依赖」
2. 等待上传完成（控制台会有提示）
3. 对以下函数重复操作：
   - `vote`
   - `comment`
   - `getPosts`
   - `feedback`

每个函数第一次部署需要在线安装 `wx-server-sdk` 依赖，预计 30 秒左右一个。

---

## 五、运行测试

### 基础流程：
1. 点击「编译」按钮（或 Ctrl+S 保存触发）
2. 在左侧模拟器中查看小程序界面
3. 点击「评测」tab → 输入物品名称 → 选择权重 → 回答问卷
4. 查看评分结果 → 发布到社区
5. 在「社区」tab 查看帖子 → 投票 → 评论
6. 在「我的」tab 管理个人资料

### 真机测试：
- 点击顶部「预览」按钮 → 手机扫码体验
- 点击「真机调试」→ 可以在电脑上调试手机上运行的小程序

## 七、常见问题

### Q: 云开发环境 ID 在哪看？
A: 微信开发者工具 → 顶部云开发按钮 → 控制台顶部显示「环境 ID」。

### Q: 云函数部署失败？
A: 确认已开通云开发。如果依赖安装失败，右键选择「上传并部署：云端安装依赖（不上传 node_modules）」。

### Q: TabBar 图标不显示？
A: 确认 `miniprogram/images/icons/` 目录下有对应的 PNG 文件。

### Q: 小程序真机无法访问数据？
A: 检查云开发控制台的数据库权限设置，确保集合权限不是「仅管理员可读写」。
